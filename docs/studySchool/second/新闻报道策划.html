                                    # 课程简介

                                    专业核心课程，具有 的特点

                                    掌握各种类型的，如突发事件报道策划、典型报道策划、会议报道策划等

                                    # 课程考核

                                    • 综合成绩=平时成绩(10%) + 过程性考核(40%) + 期末考试(50%)

                                    • 期末考试：考查(作品，策划书及版样)

                                      • 根据背景材料，自拟主题，进行新闻策划书撰写，以及版样的落实

                                    # 课程导论

                                    # 一、新闻编辑的产生、发展(关键词)

                                    进入，广播电视、互联网先后问世，这些媒介新闻编辑的分工和运作，既，显示出新的生机与活力

                                    # 二、新闻编辑工作的三个方面

                                    • 采写与编辑是新闻传播的主要环节和基本业务
                                    1. 新闻传播的主导性工作
                                      • 报道的策划
                                        • 编辑人员安排版面、节目、网页，对记者的报道给予一定的指导
                                          • 指导：主题、题材、角度、采访线索
                                      • 报道的定位
                                        • 从整体上对新闻报道进行正确的定位
                                          • 定位：方向、特色、质量
                                    2. 新闻传播的整合性工作
                                      • 新闻传播是集合传播，新闻报道刊播之前，通常要经过整合，以“产品组合”的形式提供给受众
                                      • 各种报道组合在一起，在引导受众方面具有重要作用
                                    3. 新闻传播的把关性工作
                                      • 新闻报道能否刊播的最终决定者

                                    # 三、新闻策划与组织：

                                    # 如何解决采编一体化

                                    • 建立有效的策划机制
                                      • 理顺采编关系
                                      • 提高报道质量

                                    # 一：新闻报道策划的概述

                                    # 一、

                                    # 争论源头

                                    20世纪90年代初，当策划被引入新闻报道中时，是以新闻策划的面目出现的

                                    • 新闻的定义
                                    • 新闻的体裁
                                    • 某种重要的信息
                                    # 含混的概念
                                    • 新闻策划是一个含混的概念
                                      1. 对新闻报道的策划（争议点）
                                        • 新闻工作者，以自己的主观努力促其圆满和完善，之后再予以报道
                                      2. 对新闻事件的策划
                                        • ，新闻工作者对如何报道事件进行策划，更全面、生动、深入反映客观事实
                                      3. 新闻事件的策划的要求
                                        • 不违背事物发展的规律
                                        • 不违背新闻报道的一般规律

                                    # 二、对新闻报道策划的界定

                                    • 新闻策划主要是指**的策划，或者说是**的策划，新闻策划只是一个简单扼要的提法

                                    # 三、新闻报道策划的内涵

                                    • 围绕某个主题或话题，对新闻采编活动进行有创意的**，通过指定并实施方案**，来强化传播效果以形成和引导舆论的 行为

                                    # 四、新闻报道策划的类型

                                    • 满足受众需求的策划
                                    • 战胜竞争对手的策划
                                    • 展示自家所长的策划

                                    # 五、新闻报道策划的积极作用

                                    • 有利于开发人力资源
                                    • 有利于获得最大效益
                                    • 有利于挖掘新闻资源
                                    • 有利于提高管理水平

                                    # 二：新闻报道策划的宏观层次

                                    # 一、策划的基本原则

                                    • 真实性原则
                                      • 具体真实和整体真实的统一
                                    • 导向性原则
                                      • 防止低俗化报道和**“小题大做”式报道**
                                    • 创新性原则
                                      • 创新是策划的核心和灵魂
                                    • 可行性原则
                                      • 前瞻性原则
                                        • 选题的可行性
                                        • 实施的变通性
                                        • 策划的周密性
                                        • 结果的时效性

                                    # 二、报道的定位

                                    • 报道定位的依据
                                      • 媒体自身
                                      • 受众对象
                                    • 报道定位的要领
                                      • 坚持正确导向
                                      • 满足受众需求
                                      • 体现自身特色

                                    #

                                    • 发现新闻线索与确立报道目标
                                      1. 新闻线索
                                        • 什么样的新闻线索值得挖掘
                                        • 面对众多客观事实，媒介着重报道什么
                                      2. 收集信息
                                        • 报道客体的信息
                                        • 报道接受者的信息
                                        • 报道竞争的信息
                                      3. 选题事实标准
                                        • 选题标准 —— 报道选题策划的共性规律
                                          • 时效性
                                          • 新鲜性
                                          • 重要性
                                          • 接近性
                                          • 趣味性
                                          • 冲突性
                                          • 社会情绪
                                      4. 选题个性标准
                                        • 报道选题策划的个性化
                                          • 媒介自身
                                          • 受众群体

                                    # 三：新闻报道策划的中观层次

                                    # 一、新闻单元

                                    # 新闻单元与新闻报道活动

                                    1. 定义：指的是新闻媒介产品中以传播新闻为主要职能的那些组成部分

                                      • 报纸的要闻版和其他新闻版

                                      • 网络媒介的新闻网页

                                      • 广播电视的新闻频道及新闻性栏目

                                      • 新闻性的专刊、专版

                                    2. 新闻传播策划与运行关系图

                                    # 媒介新闻单元设计

                                    1. 媒介产品走势：规模扩张，结构变多

                                    2. 4C（20世纪90年代）取代 4P（20世纪60年代）

                                      • 4C

                                        • 消费者(Consumer)

                                        • 成本(Cost)

                                        • 便利性(Convenience)

                                        • 传播(Communication)

                                        • 4P

                                          • 产品(Product)
                                          • 价格(Price)
                                          • 渠道(Place)
                                          • 推广(Promotion)

                                    3. 树状结构的新闻单元设计

                                      • 报纸的结构：
                                        • 单元 —— 版面 —— 专栏 —— 稿件
                                      • 广播电视节目的结构
                                        • 频道 —— 栏目 —— 子栏目 —— 单个节目
                                      • 网络媒体的结构
                                        • 主页目录(第一层网页) —— 专栏及稿件目录(第二层网页) —— 稿件正文(第三层网页)

                                    4. 新闻单元的定性和定量设计

                                      • 确定报纸新闻版、广播电视新闻栏目、网络媒介细纹网页在整个媒介产品汇总所占的比重
                                      • 确定报纸新闻版在报纸上的位置、出版频率和具体事件、广播电视新闻栏目播出的具体时间和播出频率、网络媒介新闻网页的位置

                                    # 二、专栏

                                    • 应社会需求产生的，媒体为了刊播某一类大致相同内容的稿件或节目而形成的拥有较为固定的名称和位置的版面或时间

                                    # 一、分类：

                                    1. 新闻专栏

                                      • 所占比例最大，引起的关注度也最大

                                    2. 评论专栏

                                      • 媒体上开设的针对某一新闻事件或现象进行评论的言论性专栏，以报纸刊发的较多

                                    3. 服务专栏

                                      • 不受时间限制，也不管事件大小，特别能体现对受众的重视，因而开设得较为普遍而且稳定

                                        为受众服务是媒体存在和发展的一个重要基点，离开了为受众服务，媒体所做的一切都是无意义的

                                    4. 个人专栏

                                      • 在报纸中常见

                                      • 最大共同点是：专栏的撰稿人有一定的知名度，因而能引起读者的关注

                                    5. 访谈类专栏

                                      • 广播、电视、平面媒体邀请一些专家和政府部门负责人到媒体编辑部或演播室共同就某一话题进行面对面的交流，使新闻更加吸引受众

                                    6. 舆论监督与群众呼声专栏

                                      • 为了促进我国政治文明建设步伐的加快，提高广大民众的素质，不少媒体开辟了舆论监督的专栏和反映群众呼声的专栏

                                    7. 媒体新闻与观点摘要专栏

                                    8. 科学专栏

                                      • 包括解释性、知识性、学术性专栏
                                      • 特点：解释和传播某种新成果、新知识或交流学术动态与见解
                                      • 主要采用的写作手段是解释性新闻、特稿、专论、综述

                                    # 二、定位和经营

                                    • 媒体靠专栏创特色
                                    • 持之以恒，不可半途而废
                                    • 以开放的心态不断创新
                                    • 讲究版面艺术，追求语言特色

                                    # 三、专版

                                    • 专版最大的特点是 不定期围绕某一个主题组织一个或几个整版的稿件

                                    # 一、新闻类专版的策划

                                    • 专版的时机选择

                                    • 专版的主题确定

                                      • 必须有明确集中的主题

                                    • 专版的内容框架

                                      没有主题的新闻类专版很零散，而没有新颖、深刻的新闻内容作支撑，专版的主题很难得以凸显

                                    # 二、服务类专版的策划

                                    • 专版的服务性

                                      为读者提供信息服务

                                      • 真实、正确
                                      • 全面
                                      • 具体

                                    # 四、专刊

                                    • 专刊是指 按周期出版，内容集中、有别于散发新闻的报纸版面，一般称之为 专副刊
                                    1. 专刊的受众分析

                                    • 读者的需求在发生变化

                                      • 专刊定位首要前提就是读者需求的变化

                                    • 关注目标读者的阅读兴趣

                                    1. 专刊的内容策划
                                    • 发挥专刊的新闻功能
                                    • 重点经营专刊中的专栏
                                    • 精心设计专刊的版面
