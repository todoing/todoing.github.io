                                                                                                          # 第一节：新闻采写在新闻传播活动中的作用

                                                                                                          # 一、新闻采写是新闻生产的基础和核心环节

                                                                                                          新闻是

                                                                                                          • 新近发生的事实（客观存在）
                                                                                                          • 公开传播出去的报道（“新闻生产”）

                                                                                                          新闻采写包含着新闻 与新闻 两个紧密相连的“”，即从 新闻事实到 新闻事实这一新闻传播最主要的生产流程

                                                                                                          新闻采写是新闻生产的 和 ，对新闻作品的成功与否起着决定性的作用

                                                                                                          发现新闻事实 —— 选择新闻事实 —— 呈现新闻事实

                                                                                                          # (一) 新闻采访的主要过程
                                                                                                          1. 获取 ，发现新闻
                                                                                                          2. 寻找可靠的信息源，核对新闻事实的基本要素
                                                                                                          # (二) 新闻写作的主要过程
                                                                                                          1. 按照受众需要和媒体功能定位，选择合适的报道角度
                                                                                                          2. 用受众喜闻乐见的方式呈现新闻事实

                                                                                                          # 二、新闻采写是记者思想水准、职业精神和专业技能的综合体现

                                                                                                          作为新闻生产基础和核心环节的新闻采写，如同人类社会的其他 一样，有着自身特定的 规律、规则，有着

                                                                                                          补充：受众商品论

                                                                                                          思考：新闻生产 = 一般“生产”(拟态环境)？

                                                                                                          • 不等同。
                                                                                                          • 新闻报道不仅呈现事实，
                                                                                                          • 因此新闻采写不仅是记者专业技能的体现，也是记者政治水平、思想水平和职业精神的体现
                                                                                                          # 新闻信息的价值判断

                                                                                                          1. 利益判断

                                                                                                            • 即 新闻信息给社会各阶层、组织、人群带来的利害关系评估。
                                                                                                            • 利或弊、得或失是利益判断的核心

                                                                                                          2. 性质判断

                                                                                                            • 即 新闻信息的本质评估
                                                                                                            • 是或非、对或错、罪或非罪、正面或负面、积极或消极、先进或落后、精华或糟粕、文明或愚昧等，都是性质判断的重点

                                                                                                          3. 态势判断

                                                                                                            • 即 新闻信息的走向评估。
                                                                                                            • 大到国际国内形势，小到各种商品的价格变化都需要态势判断
                                                                                                            • 顺或逆、进或退、高或低、升或降，都是态势判断所要回答的关键问题

                                                                                                          4. 意义判断

                                                                                                            • 即 新闻信息给社会方方面面带来的影响评估
                                                                                                            • 有或无、大或小、强或弱、长期或短期、全局或局部等，是意义判断的关节点

                                                                                                          # 第二节 ：当代新闻采写面临的挑战

                                                                                                          # 一、社会变革对新闻采写的挑战

                                                                                                          1. 直面不同群体利益矛盾，探索真相，发挥“减压阀”作用

                                                                                                          2. 捍卫和宣扬社会主义核心价值观

                                                                                                          # 二、传播新技术对新闻采写的挑战

                                                                                                          1. 解决专业新闻生产与“泛社会”新闻生产的矛盾

                                                                                                          2. 解决浅阅读、碎片化阅读、视觉化阅读与深度解决的矛盾

                                                                                                          # 三、经济全球化对新闻采写的挑战

                                                                                                          1. 拥有全球视野，坚守中国立场
                                                                                                          2. 用国际通用语言，说中国故事

                                                                                                          2. 有助于建立复合型的知识框架

                                                                                                          # 第一节：新闻采写的主要特征

                                                                                                          # 一、新闻采写是具有社会性的精神生产活动 (拉斯韦尔三功能说 -- 监视环境)

                                                                                                          1. 是新闻采写存在的 和
                                                                                                          2. 社会公众的 是新闻采写的 和
                                                                                                          3. 是新闻采写的

                                                                                                          # 二、新闻采写是具有专业性的传播行为

                                                                                                          1. 以 (新鲜) 为本能
                                                                                                          2. 以客观公正(真实)为准绳

                                                                                                          # 第二节：新闻采写的基本原则

                                                                                                          # 一、获取和呈现真实新闻信息的底线原则

                                                                                                          1. 实有其事
                                                                                                          2. 展示真相

                                                                                                          # 二、注重新闻传播规律和公众利益并重的价值原则

                                                                                                          • 价值原则，是一种意义赋予活动，是人们衡量客观事物“有用性”的判断依据
                                                                                                          • 任何新闻报道，都是新闻采写者基于自身的价值原则，对客观实际情况进行采集、加工、反映的结果

                                                                                                          # 三、讲究“传通”和社会效益双赢的效果原则

                                                                                                          • 效果原则，是人们对自身行为、行动功用的鉴定根据

                                                                                                          # 第三节：新闻采访与写作的关系（辩证思考）

                                                                                                          # 一、新闻采访决定新闻写作

                                                                                                          # 原因：
                                                                                                          1. 从新闻报道的来源和形成来看
                                                                                                            • 事实是第一性的，反映事实的新闻报道是第二性的
                                                                                                            • 先有事实，后有新闻报道，这两者之间的媒介是新闻采访
                                                                                                          2. 从新闻报道本身的特点和要求看
                                                                                                            • 新闻报道强调真实性，要求准确地反映客观事物
                                                                                                          3. 从新闻报道的具体过程看
                                                                                                            • 新闻稿诞生的七个步骤
                                                                                                              • 寻找新闻线索
                                                                                                              • 确定采访选题
                                                                                                              • 选择并占有材料
                                                                                                              • 理出观点
                                                                                                              • 提炼主题
                                                                                                              • 安排结构
                                                                                                              • 写作成稿

                                                                                                          # 二、新闻写作是新闻采访的升华

                                                                                                          1. 新闻写作过程是对新闻采访得来的材料认识上的升华
                                                                                                          2. 新闻写作对新闻采访起到“指导”作用

                                                                                                          # 三、新闻采访与写作的一体化

                                                                                                          • 新闻报道实践中，新闻采访为新闻写作提供报道所需的原材料，新闻写作依据采访得来的材料形成报道所需的文本
                                                                                                          • 两者相辅相成、互相牵制、共同作用，才使得新闻报道得以生成

                                                                                                          # 第一节：记者的角色与职责

                                                                                                          # 一、记者的角色

                                                                                                          • 传递新闻信息，传递党的意志

                                                                                                          # 二、记者的职责

                                                                                                          1. 采写新闻
                                                                                                          2. 研判舆情

                                                                                                          # 第二节：记者的修养与职业道德

                                                                                                          # 一、记者的修养

                                                                                                          1. “识”的修养
                                                                                                            • 眼力，远见卓识
                                                                                                          2. “学”的修养
                                                                                                            • 知识渊博，学力
                                                                                                          3. “才”的修养
                                                                                                            • 才能，精通业务
                                                                                                          4. 心理素质的修养
                                                                                                            • 坚忍的意志力
                                                                                                            • 灵活机智的应变能力

                                                                                                          # 二、记者的职业道德

                                                                                                          1. 服务公众，维护公众利益
                                                                                                          2. 忠于事实，维护新闻的真实性
                                                                                                          3. 注重舆论导向，坚守社会公平主义
                                                                                                          4. 遵纪守法，廉洁奉公

                                                                                                          # 一、信息意识

                                                                                                          3. 具有熟练掌握信息技能的自觉意识

                                                                                                          # 二、价值意识

                                                                                                          • 价值认知
                                                                                                          • 价值评价
                                                                                                          • 价值选择

                                                                                                          # 三、受众意识

                                                                                                          1. 研究受众，了解受众的兴趣、爱好
                                                                                                          2. 满足受众信息需求

                                                                                                          # 第一节：新闻事实的含义与特性

                                                                                                          # 一、新闻事实的含义

                                                                                                          1. 新闻事实是事实的一种
                                                                                                          2. 新闻事实是新近（或正在）发生、发现的事实
                                                                                                          3. 新闻事实是合乎新闻传播要求的事实

                                                                                                          # 二、新闻事实的特性

                                                                                                          1. 现实性
                                                                                                            • 在事实发生、发现的时间上，新闻事实是“现时”时态下呈现的
                                                                                                            • 在事实发生、发现的空间上，新闻事实是“现状”状态下呈现的
                                                                                                          2. 异常性
                                                                                                            • 新闻事实是不一般的、不寻常的事实
                                                                                                            • 新闻事实是非常态的、反常的事实
                                                                                                          3. 有效性
                                                                                                            • 新闻事实是能为人们提供“认知价值”的事实
                                                                                                            • 新闻事实是能为人们提供“兴趣价值”的事实
                                                                                                            • 新闻事实是能为人们提供“道义价值”的事实

                                                                                                          # 第二节：新闻事实的识别

                                                                                                          # 一、新闻事实的识别取决于记者的新闻敏感

                                                                                                          1. 新闻敏感的含义与作用
                                                                                                            1. 含义
                                                                                                              • 新闻敏感（新闻眼，新闻鼻），即合格记者所特有的职业敏感
                                                                                                            2. 作用
                                                                                                              • 能迅速发现新闻事实
                                                                                                              • 能对某些重大新闻事实作出科学的预见
                                                                                                          2. 新闻敏感从何而来
                                                                                                            • 高度的职业责任感
                                                                                                            • 对形势以及党和政府的方针、政策的透彻了解
                                                                                                            • 对实际情况、知识、资料的积累与研究

                                                                                                          # 二、新闻事实的识别方法

                                                                                                          1. 变动的范围与规模
                                                                                                            • 客观存在的实际情况变动的空间
                                                                                                          2. 变化的顺序与次数
                                                                                                            • 就客观实际情况变动出现的时间顺序与频率而言
                                                                                                          3. 变动的过程与速度
                                                                                                            • 就客观实际情况变动的内在关系而言

                                                                                                          # 第三节：新闻事实的选择依据

                                                                                                          # 一、新闻价值

                                                                                                          1. 新鲜性
                                                                                                          2. 重要性
                                                                                                            • 判断事实重要性
                                                                                                              • 影响力
                                                                                                              • 知名度
                                                                                                          3. 接近性
                                                                                                            • 地理的接近性（空间距离上的接近性）
                                                                                                            • 心理上的接近性
                                                                                                          4. 趣味性
                                                                                                            • 和思想性密切相关
                                                                                                            • 融于知识性之中
                                                                                                            • 讲究人情味

                                                                                                            • 确定媒体的目标受众群
                                                                                                            • 受众定位不同，决定着媒体对新闻事实选择有不同要求
                                                                                                          2. 功能定位
                                                                                                            • 新闻事实选择的又一重要依据
                                                                                                          • 两者相辅相成、辩证统一
                                                                                                          • 注重受众定位是实现功能定位的前提和基础
                                                                                                          • 注重功能定位是满足受众定位的有力保障

                                                                                                          1. 真实、准确、客观地呈现新闻事实
                                                                                                          2. 注重舆论导向，正确引导社会舆论

                                                                                                          # 第四章：新闻采写的成果 ——

                                                                                                          # 第一节：新闻报道的主要特点

                                                                                                          # 一、及时披露新闻信息

                                                                                                          1. 讲究时间性
                                                                                                            • 时间性：事实发生或变动与公开传播之间的时间差
                                                                                                          2. 运用新闻由头保证时效
                                                                                                            • 新闻由头，也叫新闻引子，是指新闻信息报道的依据或理由，即要回答为什么要报道此消息
                                                                                                            • 新闻是就近发生或变动的事实信息的报道，及时披露新闻信息，决定了新闻报道必须交代新闻由头
                                                                                                              • 事件类新闻信息，一般借助突出事实发生或变动的时间点来显示新闻由头
                                                                                                              • 非事件类新闻信息，常常借助事实进程中的最新情况或最新动向作为新闻由头

                                                                                                          # 二、真实反映报道对象

                                                                                                          1. 准确反映
                                                                                                            • 真实是新闻的生命
                                                                                                            • 采用最权威的信息来源
                                                                                                            • 核对，核对，再核对
                                                                                                          2. 全面反映
                                                                                                            • 新闻报道的真实性，还体现在其全面反映报道对象上
                                                                                                            • 新闻报道不仅要准确地呈现事件信息，还要全面地呈现事实信息中的各种“联系”

                                                                                                          # 三、客观、公正呈现新闻事实

                                                                                                          1. 强调事实与意见分开，注重用事实说话（事实+ 意见 = 宣传）
                                                                                                          2. 强调平衡性原则，注重展示新闻事件各方面的主张
                                                                                                          3. 强调交代信息来源，注重标明信息出处

                                                                                                          补充：睡眠者效应(sleeper effect)

                                                                                                          • 强调信息来源：当一个信息会新闻报刚出来时，对受众影响最大的是信息来源；

                                                                                                          • 但过一段时间之后，受众就只会记住内容，不会记住来源

                                                                                                          # 第二节：新闻报道的构成要件

                                                                                                          # 一、新闻事实

                                                                                                          1. 新闻事实呈现离不开新闻五要素（5W）
                                                                                                            • 何时（when）、何地（where ）、何事（what）、何因（why ）、何人（who ）
                                                                                                          2. 新闻各要素的运用要灵活、突出重点

                                                                                                          # 二、报道立意

                                                                                                          1. 既要“接地气”，又要“高远”
                                                                                                            • 接地气：符合客观实际，符合新闻事实的本来面目
                                                                                                            • 高远：站得高，看得远，借助新闻事实的呈现传递正能量，传播积极向上的社会主义核心价值观
                                                                                                          2. 精心选择新闻角度
                                                                                                            • 新闻角度包括三层含义：
                                                                                                              • 选题角度
                                                                                                              • 选材角度
                                                                                                              • 表现角度

                                                                                                          # 三、文本结构

                                                                                                          • 指的是新闻报道中新闻事实叙述的逻辑，分为以下三种
                                                                                                            1. 以事实素材重要程度来排列的倒金字塔结构
                                                                                                            2. 以事件发生的时间先后顺序来排列的故事性结构
                                                                                                            3. 以事件引发的追问来排列的解读性结构

                                                                                                          eg：高铁不卖卫生巾，为啥“昆铁+”卖了超一万？铁路客服回应

                                                                                                          # 四、背景材料

                                                                                                          • 指与新闻事实形成的原因、条件等相关的材料

                                                                                                          1. 背景材料的作用

                                                                                                            1. 有助于受众全面、完整地理解新闻事实

                                                                                                            2. 有助于巧妙地表现记者的立场、观点

                                                                                                            3. 有助于凸显报道的新闻价值，引起受众的关注

                                                                                                          2. 背景材料的运用

                                                                                                            • 在新闻报道中，背景材料可以分为多种类型，归纳起来大体上有三类：

                                                                                                              1. 对比性材料 —— 用作新闻任务或事件今昔、正反、左右对比的材料

                                                                                                              2. 说明性材料 —— 用作阐释与新闻事实相关的政治、经济、文化及地理环境等的材料

                                                                                                              3. 注释性材料 —— 用作诠释、介绍新闻事实中一些不易理解的内容或者专业名词、术语等的材料

                                                                                                          # 五、新闻语言

                                                                                                          1. 新闻语言的基本要求

                                                                                                            • 具体形象

                                                                                                            • 准确鲜明

                                                                                                            • 简练生动

                                                                                                            • 通俗易懂

                                                                                                          2. 使用新闻语言应注意的问题

                                                                                                            • 坚守政治底线

                                                                                                            • 坚守法律与社会伦理底线

                                                                                                            • 正确处理好传承与创新的关系

                                                                                                          • 指新闻报道中呈现新闻事实的写作方法与技巧。
                                                                                                            • 陈述笔法

                                                                                                              • 场景特写

                                                                                                            • 着力描写生动的细节

                                                                                                            • 运用社会科学的方法来呈现新闻事实的报道样式

                                                                                                            • 精确新闻的类型和特点

                                                                                                                • 通过对社会现象的状况、过程和特征进行客观、准确地描述来报道新闻

                                                                                                                • 旨在说明产生社会现象的原因，预测事物的发展趋势或后果，探寻现象之间的因果关系，从而揭示现象为什么产生，为什么会变化

                                                                                                              • 依托互联网时代的大数据技术，数据数量和规模都很大

                                                                                                          # 第一节：新闻发现的意义

                                                                                                          # 一、 没有发现就没有新闻

                                                                                                          # 二、新闻发现是新闻采写的前提

                                                                                                          # 第二节：新闻发现的目标和依据

                                                                                                          # 一、新闻发现的目标

                                                                                                          1. 新闻发现的目标是：在事实的海洋中，找出新鲜的、真实的、有新闻价值的事实，并根据这个事实发现或创造出新闻作品的最佳表达方式
                                                                                                          2. 新闻发现的目标首先在于
                                                                                                          3. 新闻发现的目标还在于 发现或创造出新闻作品最佳的表达方式

                                                                                                          # 二、新闻发现的依据

                                                                                                          • 作为新闻采写者对事实信息的传播价值及其最佳表达方式的率先认知，新闻发现实质是一种新闻价值评价
                                                                                                          • 新闻发现的依据，就是新闻价值评价的标准

                                                                                                          # 第三节：新闻发现的动态过程

                                                                                                          # 一、记者的**“前发现”状态**

                                                                                                          • 就是记者的新闻素养所营造的心理背景系统
                                                                                                          • “前发现”状态不仅决定记者能否发现新闻事实，而且决定着记者的报道取向

                                                                                                          # 二、新闻发现的关键节点

                                                                                                          • 记者新闻发现的思维过程，可分为三个阶段：
                                                                                                            • 准备阶段
                                                                                                            • 灵感阶段
                                                                                                              • 新闻发现的关键阶段
                                                                                                            • 印证阶段

                                                                                                          # 三、新闻发现的实践情境

                                                                                                          • 从空间情境、时间情境、情感情境三个维度分析：
                                                                                                            1. 不同空间情境对新闻发现的影响
                                                                                                            2. 不同时间情境对新闻发现的影响
                                                                                                            3. 不同情感对新闻发现的影响

                                                                                                          # 第四节：新闻发现的方法

                                                                                                          # 一、预设与求证

                                                                                                          • 预设：预测事实真相 + 预设传播价值（报道主题）
                                                                                                          • 求证：求证事实真相 + 确认传播价值（报道主题）

                                                                                                          # 二、寻觅悟性事实

                                                                                                          • 悟性事实：既生动形象又包含意义和价值，易使受众产生“感悟”的事实
                                                                                                          • 悟性事实必须首先是感性事实，具象而非抽象

                                                                                                          # 三、首次发现与再次发现

                                                                                                          • 指记者在完成一次报道任务时，思维多次变化与升华，对报道主题、素材和表达方式不断产生新发现的过程
                                                                                                          • 也指记者面对某些重要事实，不断开掘新的传播价值，不断发现并形成新的报道的思维过程

                                                                                                          # 第一节：新闻选题与采访策划的内涵及其作用

                                                                                                          # 一、新闻选题的内涵作用

                                                                                                          1. 新闻选题的内涵
                                                                                                            • 新闻选题分类
                                                                                                              • 从社会领域：政治类、经济类、文化类、教育类、社会类、娱乐类
                                                                                                              • 从事实的类型：动态事实选题、深度事实选题、人物事实选题
                                                                                                              • 从事实本身的属性：正面报道选题、反面报道选题
                                                                                                            • 新闻选题的确定可以从以下几方面实施：
                                                                                                              • 同一事实的不同侧面
                                                                                                              • 同一事实的不同阶段
                                                                                                              • 同一事实的不同层面
                                                                                                              • 纵向串连事实
                                                                                                              • 横向串连事实
                                                                                                          2. 新闻选题的作用
                                                                                                            • 与媒体的社会功能密切相关
                                                                                                            • 与媒体的影响力密切相关

                                                                                                          # 二、采访策划的内涵与作用

                                                                                                          1. 采访策划的内涵
                                                                                                            • 采访策划也称报道策划
                                                                                                            • 大致包含三个环节：
                                                                                                              • 报道思想的策划
                                                                                                              • 报道内容的策划
                                                                                                                • 采访项目的策划
                                                                                                                • 具体采访目标的策划
                                                                                                              • 报道体裁的策划
                                                                                                          2. 采访策划的作用
                                                                                                            • 有利于媒体生产出具有强传播力的高质量的新闻作品，提升媒体的市场竞争力
                                                                                                            • 有利于提高记者的新闻报道专业水准和职业创造力，增强其职业成就感
                                                                                                          3. 采访策划的误区
                                                                                                            • 采访策划是对报道的设计与谋划，而不是为了报道而对事实的发生进行设计和创造

                                                                                                          # 一、筛选事实与搜寻事实

                                                                                                          1. 筛选事实
                                                                                                          2. 搜寻“目标事实”

                                                                                                          # 二、选择报道角度

                                                                                                          • 可供选择的角度：
                                                                                                            • 事实的不同侧面
                                                                                                            • 事实的不同层面
                                                                                                            • 事实的表面现象、直接原因、更深层次的背景
                                                                                                            • ...

                                                                                                          # 三、关注报道效果

                                                                                                          • 预测报道将产生的社会效果，是衡量选题价值的首要条件

                                                                                                          # 四、明确报道体裁

                                                                                                          • 报道体裁的确定，取决于报道对象

                                                                                                          # 第三节：采访策划的原则与方法

                                                                                                          # 一、采访策划的基本原则

                                                                                                          1. 遵循事实第一的原则
                                                                                                            • 真实性是新闻报道的基础，无论什么样的题材，什么样的策划，都不能违背新闻真实性的原则
                                                                                                          2. 遵循传播规律的原则
                                                                                                            • 时机性
                                                                                                              • 采访策划必须重视报道推出的时机
                                                                                                            • 独家性
                                                                                                            • 现场性
                                                                                                            • 典型性
                                                                                                              • 只有新近发生的、典型的事实才是最有说服力的
                                                                                                            • 可行性
                                                                                                              • 要求采访策划的各种方案，必须是在新闻采写实践中可以实施的

                                                                                                          # 二、采访策划的主要方法

                                                                                                          1. 报道主题的策划方法

                                                                                                            • 不断提炼直至“尖准”
                                                                                                            • 不断变换角度直至“新颖别致”

                                                                                                          2. 获取报道素材的策划方法

                                                                                                            • 步骤：

                                                                                                              1. 采访者大胆想象最佳报道素材

                                                                                                              2. 通过各种采访方法去印证想象中的素材

                                                                                                          # 第七章：采访的类型

                                                                                                          • 根据记者的 ，将新闻采访分为四大类型：

                                                                                                            • 访问

                                                                                                            • 观察

                                                                                                            • 文献采集

                                                                                                            • 其他采访

                                                                                                          # 第一节：访问

                                                                                                          # 一、 当面采访

                                                                                                          1. 当面采访的优势
                                                                                                            • 最可靠
                                                                                                              • 便于确认身份
                                                                                                              • 便于取得互信
                                                                                                              • 便于去伪存真
                                                                                                            • 当面采访的其他优势
                                                                                                              • 能为观察创造条件
                                                                                                              • 能获得丰富信息，发现新线索
                                                                                                              • 能为进一步交往打下基础
                                                                                                          2. 当面采访的形式
                                                                                                            • 常见形式
                                                                                                              • 事先约定的采访
                                                                                                              • 临时遭遇的采访
                                                                                                              • 一对多的采访
                                                                                                              • 新闻现场的采访
                                                                                                            • 集体采访也是当面采访的一种
                                                                                                              • 各类新闻发布会
                                                                                                              • 对某些“重大题材”或“重大典型”的集体采访
                                                                                                              • 媒体合并采访

                                                                                                          # 二、电话采访

                                                                                                          • 电话采访是指记者借助电话等通信工具与被采访者交流与对话的访问方式

                                                                                                            1. 电话采访的优势

                                                                                                              • 便捷

                                                                                                              • 高效

                                                                                                              • 低耗

                                                                                                              • 有现场感

                                                                                                            2. 电话采访的设备

                                                                                                              • 固定电话
                                                                                                              • 手机
                                                                                                              • 卫星电话
                                                                                                              • 网络电话

                                                                                                            3. 电话采访的弊端

                                                                                                              • 不利于互信
                                                                                                              • 可能产生交流困难
                                                                                                              • 可能产生理解错误
                                                                                                              • 可能被广告电话误伤
                                                                                                              • 依赖设备

                                                                                                          # 三、网络采访

                                                                                                          • 也叫电子采访，是在以互联网为基础的新媒体兴起以来最受欢迎、也相当实用的新型采访形式
                                                                                                          • 其优越性随着新闻传播的数字化发展和信息技术的进步愈发明显
                                                                                                          1. 网络采访的优势
                                                                                                            • 快捷传递
                                                                                                            • 精准表达思想
                                                                                                            • 大容量和多媒体
                                                                                                            • 多元化采访 （一对多）
                                                                                                          2. 网络采访的形式
                                                                                                            • 电子邮件
                                                                                                            • 在线对谈
                                                                                                            • 社交媒体
                                                                                                          3. 网络采访的局限
                                                                                                            • 难辨身份
                                                                                                            • 较为被动
                                                                                                            • 受限于环境

                                                                                                          # 第二节：观察

                                                                                                          • 观察作为记者对客观事物由表及里进行察看和感知的主动行为，也是新闻采访的一种重要类型

                                                                                                          # 一、场景观察

                                                                                                          • 场景观察：指作者在新闻现场利用五官主动感受事物的过程

                                                                                                          # 二、人物观察

                                                                                                          • 特指对新闻人物或采访对象的表情态度、言谈举止乃至着装穿戴、随身用品等的观察，用以揭示人物的个性特征、内心世界、人物与新闻事件的关系等

                                                                                                          # 三、细节观察

                                                                                                          • 特指采访中有意识地对现场事物或人物活动的某些细节留意、跟踪、捕捉以获取特有的新闻含义

                                                                                                          # 第三节：文献采集

                                                                                                          • 文献采集也是新闻采访的一种类型，是记者获取信息必不可少的重要手段
                                                                                                          • 文献采集属于间接采访，即在不与采访对象接触的情况下

                                                                                                          # 一、已出版文献资料的采集

                                                                                                          • 已出版文献资料：指各类正式出版的文献，包括图书、期刊、报纸、影音出版物等

                                                                                                          # 二、非出版文本资料的采集

                                                                                                          • 非出版文本资料，指未正式出版（无出版书号或刊号），由机构或个人印刷复制或手写的文本，记者习惯上统称为“资料”
                                                                                                          1. 资料在采访中的作用
                                                                                                            • 资料作为文字依据，可用于核实和验证访谈中获得的信息，包括证实和证伪
                                                                                                            • 资料可以丰富采访内容，提供背景资料和新的线索
                                                                                                            • 资料可为今后的跟进采访或扩大采访带来方便
                                                                                                          • 资料不等于事实，记者不可不核查

                                                                                                          # 三、会议材料的采集

                                                                                                          • 会议材料指记者采访会议时围绕采访主题搜寻获取的文件和影音材料，包括正式和非正式出版物

                                                                                                          # 四、网络信息的检索

                                                                                                          1. 通过搜索引擎
                                                                                                          2. 通过目标网站检索
                                                                                                          3. 通过社交网站检索
                                                                                                          4. 通过工具书类网站检索

                                                                                                          # 第四节：其他采访

                                                                                                          # 一、体验式采访

                                                                                                          • 又称为参与式采访

                                                                                                          • 是记者以获取新闻为目的，亲身参与某项工作、事件、活动等的一种采访类型

                                                                                                          • 也是主体（记者）最大限度接近客体（新闻事实）的新闻职业行为

                                                                                                          • 体验式采访在特稿和深度报道中较为常见

                                                                                                          # 二、隐式采访

                                                                                                          • 又称隐蔽性采访，俗称“暗访”
                                                                                                          • 是一种记者隐匿了身份而有效接近真相的新闻采访活动，它与亮明记者身份的“显性采访”相对应
                                                                                                          • 三大基本特征
                                                                                                            • 隐瞒记者身份
                                                                                                            • 采访对象不知
                                                                                                            • 秘密获取信息

                                                                                                          Tip：隐秘采访的目的，在于实施显性采访难以实施的采访行为，获取显性采访无法获得的新闻信息，因而是一种不得已而为之的采访。也正因此，隐性采访具有难以替代的重要作用，是有助于还原真相的特殊新闻职业行为

                                                                                                          • 隐形采访的弊端
                                                                                                            • 具有一定的危险性
                                                                                                            • 面临法律和社会伦理的困境
                                                                                                            • 可能侵犯隐私
                                                                                                            • 可能有违其他法律问题
                                                                                                            • 可能被滥用

                                                                                                          # 三、社会调查

                                                                                                          • 作为一种社会科学的调研方法，其旨在通过获取面上的数据和广泛意见，以排除个别和偶然因素，更好地体现新闻的客观、全面、公正，增强新闻的客观性和说服力
                                                                                                          • 社会调查通常包括：
                                                                                                            • 全面调查
                                                                                                            • 局部调查
                                                                                                            • 抽样调查（在新闻采访实践中，最常用）
                                                                                                          • 新闻采访中的社会调查主要形式
                                                                                                            • 问卷调查
                                                                                                            • 媒体调查
                                                                                                            • 电话调查
                                                                                                            • 合作调查

                                                                                                          补充： 李克特五度量表：符合，非常符合，比较符合，一般符合，非常不符合

                                                                                                          # 第九章：采访的实施 —— 访问

                                                                                                          • 访问是一种通过记者与采访对象面对面交谈、提问等来进行采访的 社会活动

                                                                                                          # 第一节：访问的特质

                                                                                                          # 一、访问 是一种特殊的 社会交往活动

                                                                                                            • 访问目的是为了 满足公众的信息需求 ，是一种 公共而非个体因素驱动 的交往行为
                                                                                                            • 记者的访问是一种 职业行为，其身份是 所在媒体和公众 的代表
                                                                                                            • 访问全过程 由记者主导和控制

                                                                                                          # 二、记者和采访对象的关系

                                                                                                          • 地位平等
                                                                                                          • 互有所需
                                                                                                            • 记者：为了 获得信息
                                                                                                            • 采访对象：体现了其 表达的需求
                                                                                                          • 组合自由

                                                                                                          # 第二节：访问的氛围

                                                                                                          # 一、把握访问的心理场

                                                                                                          B=f（P,E）

                                                                                                          库尔特 · 勒温 提出的心理场概念

                                                                                                          其中 B 表示行为（Behavior），f 为函数，P 表示个体（Person），E 表示环境（Environment）。即 人的行为是个人和环境的函数

                                                                                                          • 个体因素
                                                                                                          • 环境因素
                                                                                                            • 物理环境：访问的时间、地点、参与人员等
                                                                                                            • 精神环境：访问的舆论气候、组织压力等

                                                                                                          # 二、营造融洽的访问氛围

                                                                                                          • 选择合适的时间地点
                                                                                                          • 留下良好的第一印象
                                                                                                            • 遵约守时
                                                                                                            • 称呼恰当
                                                                                                            • 举止得体
                                                                                                          • 寻找相似或接近元素

                                                                                                          # 三、应对访问中的冲突

                                                                                                          • 问而不答

                                                                                                            受访者面对记者提出的问题置之不理

                                                                                                          • 答非所问

                                                                                                            受访者在采访过程中可能因为谈话太过投入，或者故意回避某些问题，而向记者提供大量与采访目的无关的信息，使对话偏离主题

                                                                                                          • 所答非实

                                                                                                            “真与假”的矛盾，也是访问过程中记者与受访者间容易出现的冲突

                                                                                                          # 第三节：提问的类型和方法

                                                                                                          # 一、提问的基本类型

                                                                                                          • 开放式提问
                                                                                                            • 所提的问题比较宽泛，能给受访者的作答以一定的伸缩余地和灵活性
                                                                                                          • 封闭式提问
                                                                                                            • 这类提问的指向具体、明确，要求一个有针对性的答案

                                                                                                          # 二、提问的一般要求

                                                                                                          • 提问要简洁
                                                                                                          • 提问要具体
                                                                                                            • 切勿笼统提问，如问一个学生“学习压力大吗？”这样的笼统问题
                                                                                                          • 提问要有逻辑
                                                                                                            • 要求记者围绕着采访目的，有层次、有针对性地展开提问，避免东拉西扯
                                                                                                            • 漏斗式：由大到小、由笼统到具体的组织问题的方式
                                                                                                            • 倒漏斗式：从小到大、由具体到一般、由微观到宏观的组织问题的方式
                                                                                                          • 提问要看环境氛围

                                                                                                          # 三、提问的主要方法

                                                                                                          • 正问法
                                                                                                            • 又叫 开门见山法
                                                                                                            • 即 记者直截了当地提出问题，是一种简单、直接的提问方法
                                                                                                          • 侧问法
                                                                                                            • 又叫 迂回法
                                                                                                            • 指 记者不直接从正面提出问题，而从侧面入手，或者绕个弯子，提些表面看来与访问内容无关的问题，以引发受访者讲出真实情况
                                                                                                          • 激将法
                                                                                                            • 又叫 反问法、故意错问法
                                                                                                            • 即 从相反方面提出问题，或者有意误解对方的意思，以刺激、激化其情绪，使其不吐不快
                                                                                                              • 用敏感问题刺激
                                                                                                              • 用错误事实刺激
                                                                                                          • 假借法
                                                                                                            • 指 记者通过假设环境或借用他人的表述作为问题引出的依据，来进行提问
                                                                                                              • 借受访者的话
                                                                                                              • 借第三者的话
                                                                                                              • 假设话题
                                                                                                          • 盘问法
                                                                                                            • 调查性报道 (揭丑报道) 中记者常用的方法
                                                                                                            • 特指围绕着某一问题不断变换发问角度或进行连续追问
                                                                                                              • 一题多问
                                                                                                              • 连续追问

                                                                                                          # 第十章：采访的实施 —— 现场观察

                                                                                                          # 第一节：现场观察的作用和内容

                                                                                                          # 一、作用

                                                                                                          • 还原事件现场

                                                                                                          • 调动意象语言

                                                                                                          # 二、内容

                                                                                                          • 观察报道对象的形貌
                                                                                                          • 观察报道对象的行动
                                                                                                            • 丰富报道的内容，而且能增强报道的可读性

                                                                                                            • 所谓环境，包括社会环境和自然环境
                                                                                                          • 观察报道事件的发展过程

                                                                                                          # 第二节：现场观察的方法和技巧

                                                                                                          # 一、方法

                                                                                                          • 观察的顺序
                                                                                                            • 总 —— 分 —— 总
                                                                                                            • 依据事件发展的过程，逐次展开对现场场景和人物言行的描写

                                                                                                          • 观察全局与观察细节并重
                                                                                                          • 用眼观察与用心去“看”
                                                                                                          • 结合背景资料进行现场观察

                                                                                                          # 一、努力抵达现场

                                                                                                          记者应该努力抵达现场，通过自己专业性的现场观察及采访获得信息并进行求证，以准确和全面地报道新闻

                                                                                                          # 二、感受现场气氛

                                                                                                          # 三、树立画面意识

                                                                                                          要求记者拥有一双擅长发现和体察独特性的眼睛，从而让报道为受众提供其他人无法复制的画面

                                                                                                          要求记者在观察中注意“五官并用”，捕捉现场的声音、气味、触感等元素并准确描述，从而为现场画面提供更为丰富的内容，让受众“感同身受”

                                                                                                          # 四、捕捉典型细节

                                                                                                          • 寻找与报道主题密切相关的典型细节
                                                                                                          • 寻找具有反差感的典型细节
                                                                                                          • 寻找具有突出画面感的典型细节
                                                                                                          • 寻找能够说明问题的数字细节

                                                                                                          # 第十一章：采访的实施 —— 记录与核实

                                                                                                          # 第一节：采访记录的作用与内容

                                                                                                          # 一、作用

                                                                                                          • 为写稿提供帮助
                                                                                                          • 防范报道带来的风险
                                                                                                            • 对于记者而言，采访记录还是一把“保护伞”
                                                                                                            • 当出现与报道相关的纠纷时，采访记录是记者最有力的证据
                                                                                                          • 记录也能独立成文
                                                                                                            • 采访记录本身也能成为独立的新闻作品
                                                                                                          • 帮助记者提高业务水平

                                                                                                          # 二、内容

                                                                                                          • 问答内容

                                                                                                            • 事件的要素信息
                                                                                                            • 重要或精彩的原话
                                                                                                            • 其他相关资料和信息

                                                                                                          • 观察内容

                                                                                                            指记者在采访中现场观察到的信息

                                                                                                          • 思考内容

                                                                                                          # 第二节：采访记录的方式与技巧

                                                                                                          # 一、方式

                                                                                                          • 用笔记
                                                                                                          • 用脑记
                                                                                                          • 用录音设备记

                                                                                                          # 二、技巧

                                                                                                          • 巧记和快记
                                                                                                          • 简记和补记
                                                                                                          • 分类记和分区记

                                                                                                          # 第三节：采访记录的整理与核实

                                                                                                          # 一、整理

                                                                                                          • 可按照 事物发生的先后顺序 进行整理
                                                                                                          • 可按照 不同的事件性质 进行整理
                                                                                                          • 可按照 不同的问题归纳 进行整理
                                                                                                          • 可按照 不同的任务特点 进行整理
                                                                                                          • 可按照 设想的稿件内容 进行整理

                                                                                                          # 二、核实

                                                                                                          • 向受访者核实
                                                                                                          • 与其他信源交叉核实
                                                                                                          • 现场调查核实

                                                                                                          # 模拟新闻发布会

                                                                                                          # 新闻发布人

                                                                                                          • 国家、政党、社会团体任命或指定的专职(比较小的部门为兼职)新闻发布人员
                                                                                                          • 其职责是在一定时间内就某一重大事件或时局问题，举行 新闻发布会或约见个别记者 ，发布新闻或阐述本部门的观点，代表有关部门回答记者的提问
                                                                                                          • 新闻发言人是新闻发布制度的重要组成部分

                                                                                                          # 新闻发布制度

                                                                                                          • 在一定事件内就某件大事件或政策问题，举行新闻发布会，或者约见记者，发布新闻或阐述所代表的政府、某个社会组织的态度立场，并且作为部门或某个社会组织的代表回答提问

                                                                                                          # 新闻发布的主体

                                                                                                          • 国务院新闻办公室
                                                                                                          • 国务院各部门（中央各部门）
                                                                                                          • 省、市、县新闻发布

                                                                                                          # 新闻发布的层次

                                                                                                          • 不定期发布

                                                                                                            • 配合党和国家有关重要 方针政策出台
                                                                                                            • 回应 突发公共事件
                                                                                                            • 社会热点问题

                                                                                                          • 定期发布

                                                                                                          # 新闻发布的策划环节

                                                                                                          • 发布主题
                                                                                                            • 政府要说的、媒体关注的、公众关心的
                                                                                                          • 发布形式
                                                                                                            • 重大活动 - 系列构成，点面结合
                                                                                                            • 突发公共事件 - 联动组合，滚动发布
                                                                                                            • 敏感社会议题 - 集中力量，多元发布
                                                                                                          • 发布人
                                                                                                            • 新闻发言人 、 相关领导 、 领域专家
                                                                                                          • 发布对象
                                                                                                            • 根据内容确定传播目的和范围（地区和人群）
                                                                                                          • 发布时机
                                                                                                            • 遵循 权威性 和 时效性 原则
                                                                                                              • 某些情况下需要考虑媒体的发稿时限；避免其他重大新闻“冲”掉所发布的新闻
                                                                                                          • 发布平台
                                                                                                          • 发布地点

                                                                                                          # 新闻发布的基本原则

                                                                                                          • 第一时间 ：黄金一小时

                                                                                                            第一时间以网站、微博、微信等形式发布简短声明表达关注，再由发言人召开新闻发布会以散发正式的新闻通告、回答记者提问的形式传递深层次的信息

                                                                                                          • 指导性信息 和 调适性信息 相结合

                                                                                                          指导性信息要准确和一致；调适性信息环节情绪，表达情感

                                                                                                          • 谣言应对

                                                                                                          • 坦诚开放式应对谣言的基本原则

                                                                                                          • “实话实说，不必劝说”

                                                                                                          • “快讲事实，慎讲原因”

                                                                                                          • 新闻循环圈
                                                                                                          • 网络社交媒体 45 分钟；广播电视 6 小时；通讯社 3 天；报纸和周刊 2 周

                                                                                                          # 新闻发布信息设计的五个基本原则

                                                                                                          • 增加与其的收益
                                                                                                          • 通过一些具体的、人性化的细节向受众阐述政府决策能够给公众带来哪些实际的收益
                                                                                                          • 减少预期的成本
                                                                                                          • 按照社会交换理论，公众往往更容易接受那些“惠而不费”的“指导性”信息
                                                                                                          • 强化正面舆论
                                                                                                          • 通过媒体平台传递正面信息，扩散正能量
                                                                                                          • 为个人创造条件
                                                                                                          • 调动公众的主观能动性，增强参与意识
                                                                                                          • 提供替代性方案
                                                                                                          • 满足不同受众的需要
                                                                                                          • 按照“高端、中端、低端”的标准提供各种方案，阐明利弊，供他们选择

                                                                                                          # 新闻发布会的简单流程

                                                                                                          1. 引领发布人进入现场
                                                                                                          2. 宣布新闻发布活动正式开始
                                                                                                          3. 分别 介绍新闻发布团队各位新闻发言人的姓名与职务
                                                                                                          4. 简单介绍即将进行发布的 主题 和相关 简单背景
                                                                                                          5. 请 新闻发言人依次发布信息
                                                                                                          6. 主新闻发言人发言完毕后，提示各新闻发言人进入提问环节，由主持人邀请记者提问
                                                                                                          7. 宣布发布会结束

                                                                                                          # 第十二章：消息写作

                                                                                                          • 消息：以简要的语言文字迅速传播新近发生的事实的新闻体裁

                                                                                                          # 第一节：消息的特点与类型

                                                                                                          # 一、特点

                                                                                                          • 事实简明

                                                                                                            • 言简意赅

                                                                                                          • 注重时效

                                                                                                            • 迅速、及时地呈现

                                                                                                          • 倒金字塔结构为主

                                                                                                            • 倒金字塔结构，就是按照重要性程度递减的顺序来编排、呈现事实

                                                                                                          # 二、消息的类型

                                                                                                          • 简讯

                                                                                                              1. 又称短讯、快讯。

                                                                                                              2. 特点：迅速快捷，争分夺秒；篇幅短小，信息单一；结构简单，不分导语主体

                                                                                                              3. 形态：一句话新闻、一段话简讯

                                                                                                          • 动态消息

                                                                                                            • 迅速及时地反映现实世界最新变动状态的消息的统称

                                                                                                            • 特点：

                                                                                                              1. 以事物的最新变动为主要着眼点
                                                                                                              2. 以客观叙事为主要特征。
                                                                                                              3. 以突发性事件为主要报道内容

                                                                                                          • 综合消息

                                                                                                            • 一般是报道发生在一段较长时间内或较大地区的某一事件、成就、趋势或问题的新闻

                                                                                                            • 特点：

                                                                                                              1. 报道面宽，气势较大；
                                                                                                              2. 点面结合，总揽全局，既有广度又有深度

                                                                                                            • 分类

                                                                                                              1. 横断面的综合

                                                                                                                • 即 对一些地区或部门所开展的某项工作、某个活动，按其范围或类别综合地呈现其全貌

                                                                                                              2. 纵深度的综合

                                                                                                                • 即 对一定时期某条战线或某项工作的成就、发展变化，按其因果关系或时空顺序加以综合反映；

                                                                                                              3. 纵横结合

                                                                                                                • 即 上述两种类型的综合运用

                                                                                                          # 第二节：消息的构成

                                                                                                          # 一、标题

                                                                                                          • 样式

                                                                                                            • 分类

                                                                                                              1. 功能：

                                                                                                                1. 主题(正题)

                                                                                                                概括与说明主要事实和思想内容

                                                                                                                1. 引题(眉题)

                                                                                                                揭示消息的思想意义或交代背景，说明原因，烘托气氛

                                                                                                                1. 副题(次题)

                                                                                                                提示报道的事实结果，或作内容提要

                                                                                                              2. 构成形式：

                                                                                                                1. 单行标题
                                                                                                                2. 双行标题
                                                                                                                3. 多行标题

                                                                                                          • 不同标题体现不同的侧重点

                                                                                                          补充：消息头
                                                                                                          • 消息头是消息的标志
                                                                                                          • 分为“讯”和“电”两类
                                                                                                            • 电：外地来电，一般为非本地的消息来源
                                                                                                            • 讯：本地媒体/报社自己获得的消息
                                                                                                          • 作用
                                                                                                          1. 使消息明显区别于其他文体
                                                                                                          2. 表明消息来源，利于读者判断
                                                                                                          3. 版权所有
                                                                                                          4. 关乎新闻发布单位的声誉，迫使新闻发布单位谨慎地对待每一条新闻

                                                                                                          # 二、导语

                                                                                                          • 导语是消息开头用来提示新闻要点与精华、发挥导读作用的段落
                                                                                                          • 一般由最新鲜、最重要的新闻事实或依托新闻事实的精辟议论组成
                                                                                                          • 特征：
                                                                                                            • 消息体裁所特有的
                                                                                                            • 位于消息的开头
                                                                                                            • 提炼了新闻事实的最重要的部分
                                                                                                            • 作用在于吸引受众阅读（收听、收看）
                                                                                                          补充导语类型
                                                                                                          • 叙述型导语：最常见，最好用的类型，高度概括提炼最重要的事实
                                                                                                          • 描述型导语：白描，对新闻事实中一个有意义的侧面、场景进行描写，具有可视感
                                                                                                          • 结论式导语：将新闻事实的结论在开头部分写出来，开门见山
                                                                                                          • 评论式导语：对报道的事实进行简介、精辟的评论，以解释事物的性质和作用，引起读者重视
                                                                                                          • 引导式导语：用采访对象的话，直接或间接引语，增强导语的主动性
                                                                                                          • 设问式导语：用提问的方式引出报道事实，设置悬念，引起读者的注意和思考
                                                                                                          • 悬念式导语：用让人惊讶的元素把读者吸引到报道中
                                                                                                          • 橱窗式导语：写有代表性的故事，读者可以推而广之想见全貌

                                                                                                          # 三、主体

                                                                                                          • 又称为“新闻躯干”
                                                                                                          • 是消息的中心部分
                                                                                                          • 主体担负着两大任务
                                                                                                            1. 解释和深化导语
                                                                                                            2. 补充导语没有涉及的新事实

                                                                                                              常见问题：不分段、逻辑混乱、材料堆砌、语句不通顺、面面俱到

                                                                                                          # 四、背景

                                                                                                          • 消息中对新闻事实进行解释、补充和烘托的材料

                                                                                                          • 作用:

                                                                                                            1. 提供知识，答疑解惑
                                                                                                            2. 补充新闻事实，突出新闻特点，提升新闻价值
                                                                                                            3. 代替记者的议论，表明、暗示记者的观点

                                                                                                          # 五、结尾

                                                                                                          • 消息最后收束全文的部分

                                                                                                          • 常见类型：

                                                                                                            1. 拾遗补缺
                                                                                                              • 或补充新闻事实、背景材料，或回应导语
                                                                                                            2. 画龙点睛
                                                                                                              • 概括中心，突出主题
                                                                                                            3. 别开生面，借题发挥

                                                                                                          # 第三节：导语的写作

                                                                                                          # 导语的分类

                                                                                                          • 按新闻要素

                                                                                                            1. 侧重人物要素的导语
                                                                                                            2. 侧重事件要素的导语
                                                                                                            3. 侧重时间要素的导语
                                                                                                            4. 侧重地点要素的导语
                                                                                                            5. 侧重原因要素的导语
                                                                                                            6. 侧重结果要素的导语

                                                                                                          • 按表达形式

                                                                                                            1. 叙述型导语
                                                                                                            2. 描写型导语
                                                                                                            3. 议论型导语

                                                                                                          # 导语的写作要求

                                                                                                          • 遵循三大原则
                                                                                                            1. 最新原则
                                                                                                            2. 吸引受众原则
                                                                                                            3. 最简原则

                                                                                                          # 导语的写作技巧

                                                                                                          • 开门见山
                                                                                                          • 设置悬念
                                                                                                          • 化静为动
                                                                                                          • 以小见大

                                                                                                          # 第四节：背景的写作

                                                                                                          # 一、分类：

                                                                                                          • 烘托对比性背景材料
                                                                                                            • 用来与新闻事实构成对比关系，从某方面烘托新闻事实
                                                                                                          • 说明性背景材料
                                                                                                            • 用来说明新闻事实的由来，对新闻事实产生的相关条件进行交代
                                                                                                          • 注释性背景材料
                                                                                                            • 对概念、术语、著名历史事件和人物、有关科学知识进行解释的文字

                                                                                                          # 二、写作要求

                                                                                                          • 精心选择背景材料
                                                                                                          • 灵活运用背景材料

                                                                                                          # 三、写作技巧

                                                                                                          • 开门见山，铺陈渲染
                                                                                                          • 正反对比，凸显个性

                                                                                                          # 第五节：主题的写作

                                                                                                          # 一、结构

                                                                                                          • 按事实重要性顺序安排结构
                                                                                                            • 倒金字塔结构
                                                                                                          • 按时间先后顺序
                                                                                                            • 顺序和倒叙
                                                                                                          • 按逻辑关系
                                                                                                            • 主次、因果、并列、点面关系
                                                                                                          • 时间顺序和逻辑关系结合
                                                                                                            • 纵向和横向

                                                                                                          # 二、写作要求

                                                                                                          • 展开导语要素，解释和深化新闻导语
                                                                                                          • 补充导语中未出现的新闻要素，将概括的事实具体化
                                                                                                          • 未受众解惑，满足受众的接受心理
                                                                                                          • 增添细节性材料，以增强消息的可读性

                                                                                                          # 三、写作技巧

                                                                                                          • 根据新闻事实的特点组织材料
                                                                                                          • 提行分段，层次分明
                                                                                                          • 语言生动，注重可读性

                                                                                                          # 第十三章：通讯写作

                                                                                                          通讯的概念
                                                                                                          • 运用多种表现方法比较深入而又详细地报道真实的客观事物的新闻文体。主要的报道范围是新闻事件的发展过程、新闻人物的成长经过、某项工作的详细情况及某地风貌的概况特色等，是最常用的报道形式之一

                                                                                                          # 第一节：通讯的特点

                                                                                                          1. 细化报道内容

                                                                                                            • 与消息不同，通讯对有关人、事、景、物的叙写较为具体和舒展，会将新闻事实的细节以及相关的历史事实（背景材料）叙述得很详细。

                                                                                                          2. 讲究主题表达

                                                                                                          详情
                                                                                                          • 主题是作者通过所写内容表达的统摄全篇的思想观念，它是整个文本的灵魂，是全部素材和文字的凝聚中心。
                                                                                                          • 通讯必须有鲜明的主题，否则就会导致两个后果：一是文本失之松散，二是立意失之肤浅。
                                                                                                          • 通讯十分讲究主题的表达：叙事记述型通讯，通常通过对人和事的叙述表达主题；调查分析型通讯，往往在陈述调查所得材料的基础上，借助对材料的分析表达主题；谈话实录型通讯，其主题则在谈者与访者的交谈中得以呈现。
                                                                                                          1. 注重讲述故事
                                                                                                          详情
                                                                                                          • 通讯是以讲故事见长的新闻文体。
                                                                                                          • 在通讯中讲故事的前提是：
                                                                                                          • 故事具有新闻性；
                                                                                                          • 故事服从于表现主题的需要，围绕主题展开。

                                                                                                          • 这就要求记者在采访时注意发现具有新闻价值的故事，再将其写进新闻报道中。

                                                                                                          1. 讲生动有趣的故事。
                                                                                                          2. 讲意味深长的故事。
                                                                                                          3. 讲感人至深的故事。

                                                                                                          【案例】《兵团日报》通讯 《“我要做一个诚信的人”》（2008 年 11 月 13 日），讲述了吴兰玉老人拾荒 9 年积攒微薄收入还债的故事。这个年迈、孱弱、贫困的老 太太所演绎的故事，凸显出的诚信，正是如今一些社会组织和个人所欠缺的。故事发人深省、催人振奋。

                                                                                                          1. 展现生动细节
                                                                                                          详情
                                                                                                          • 注重生动细节的描绘，是通讯区别于消息的又一主要特点。
                                                                                                          • 细节是指人物、事件、场景中有意味的细微之处。
                                                                                                            • 它有时是人物的举手投足，有时是事件的一个细小情节，有时是关于人物、事件的一个镜头。

                                                                                                          【案例】《县委书记的榜样——焦裕禄》（《人民日报》1966 年 2 月 7 日），其中写到这样的细节：“右脚踩在椅子上，用右膝顶住肝部”“左手经常揣在怀里”“用一根硬东西顶在右边的椅靠上”“右边被顶出了一个大窟窿”……

                                                                                                          1. 表达方式多样
                                                                                                          • 叙述 > 描写 > 说明
                                                                                                          • 议论和抒情受到较严格控制。
                                                                                                          详情
                                                                                                          • 记者在写消息时，用得最多的表达方式是叙述，描写和说明次之；议论和抒情则受到较为严格的控制。
                                                                                                          • 记者不宜直接站出来进行议论；一般来说也不宜抒发自己的情感，否则会使报道带上比较浓厚的主观色彩。
                                                                                                          • 而在通讯中，上述五种表达方式往往是同时并用的，呈现出表达方式的多样性。

                                                                                                          # 第二节：通讯的类型和解构

                                                                                                          1. 叙事记述型通讯及其结构
                                                                                                            • 这类通讯，以叙事的方式记人、记事、记地

                                                                                                              （分别对应通常所说的人物通讯、事件通讯、风貌通讯）。

                                                                                                            • 使用比较多的表达方式是叙述和描写
                                                                                                            • 其通常的结构方式有：
                                                                                                            1. 纵式结构
                                                                                                            2. 横式结构
                                                                                                            3. 纵横交叉式结构
                                                                                                            4. 其他结构
                                                                                                          通常的结构方式

                                                                                                          1. 纵式结构

                                                                                                            • 按时间顺序安排结构
                                                                                                            • 按逻辑层次安排结构

                                                                                                          2. 横式结构

                                                                                                            • 截取某个时间点或跨度较小的时间段，由此对不同空间中的人和事进行报道。
                                                                                                            • 从逻辑关系上体现各个构成部分的并列存在和平行推进。

                                                                                                          3. 纵横交叉式结构

                                                                                                            • 这种报道通常在以时间为纵轴叙事的过程中，注重其间某些时间点沿着空间的横轴延伸。

                                                                                                          4. 其他结构

                                                                                                          1. 调查分析型通讯及其结构
                                                                                                          • 特点:
                                                                                                            1. 注重问题意识
                                                                                                            2. 就问题进行分析探究
                                                                                                          详情

                                                                                                          • 调查分析型通讯是记者对现实生活中存在的问题进行调查和分析后写成的通讯

                                                                                                          • 这类通讯所涉及的，以负面题材居多，也正因为如此，它有着无可取代的舆论监督作用。

                                                                                                          • 其特点主要有：

                                                                                                            • 一是注重问题意识。
                                                                                                            • 二是就问题进行分析探究。

                                                                                                          • 结构：

                                                                                                            1.顺藤摸瓜型
                                                                                                            • 顺藤摸瓜，即由结果出发去追溯原因。
                                                                                                            • 通讯所涉及的事件结果已经显现出来并广为人知，或者是虽并未浮出水面但已经被具有新闻敏感的记者捕捉到，直奔这些引起现有结果的原因而去采访，便是顺藤摸瓜型的调查分析。
                                                                                                            2.内容归类型
                                                                                                            • 这种类型的通讯，通常围绕一个问题展开调查，调查的内容涉及问题的方方面面。
                                                                                                            • 该类通讯的若干部分之间是相互并列的板块关系，几个板块组成了一个有机整体。
                                                                                                            • 注重内容之间既可以相对独立又存在着紧密联系的逻辑关系。
                                                                                                          1. 谈话实录型通讯及其结构
                                                                                                          • 谈话实录型通讯，是以记录人物谈话方式写成的通讯。
                                                                                                          • 谈话实录型通讯的结构，通常都是采用一问一答式。
                                                                                                            详情
                                                                                                            • 谈话实录型通讯，是以记录人物谈话方式写成的通讯。
                                                                                                            • 在上述两类通讯中，记者也会记录人物（报道对象或除此以外的采访对象）的谈话，但细加比较，谈话实录型通讯与它们有所不同：
                                                                                                              • 其一，谈话实录型通讯由人物谈话充当“主角”。
                                                                                                              • 其二，谈话实录型通讯中，记者的提问以及采访对象所说的内容被大量引入文本。
                                                                                                            • 谈话实录型通讯的结构，通常都是采用一问一答式。

                                                                                                          # 第三节：通讯的主题与表达

                                                                                                          1. 通讯的主题
                                                                                                          详情
                                                                                                          • 对通讯主题的要求
                                                                                                            1. 确保正确
                                                                                                            2. 务必新颖
                                                                                                            3. 力求深刻
                                                                                                          • 通讯主题的确立和深化
                                                                                                            • 通讯主题，需要记者在采访所得材料的基础上确立
                                                                                                            • 也就是说，主题受制于采访以及从采访中获得的素材。
                                                                                                            • 不仅如此，记者还需要在不断深化对素材的认识的过程中深化主题。
                                                                                                          1. 通讯的选材
                                                                                                          详情
                                                                                                          • 选材和材料
                                                                                                            • 选材，即选取题材和素材，题材和素材统称为材料。
                                                                                                            • 题材是指社会生活某些领域的内容（如工业题材、农业题材等），可分为历史题材和现实题材。
                                                                                                            • 通讯属于新闻文体，被选入通讯的题材，通常是现实题材。
                                                                                                            • 素材是指记者在采访中获得的具体的“原材料”。
                                                                                                              • 题材、素材都有正面和负面之分，不是说一定要正面的题材和素材才能写进通讯，但必须确保写进通讯的题材和素材应能产生良好的正面效果。
                                                                                                          • 选材的具体方法
                                                                                                            1. 围绕主题选材
                                                                                                            2. 选取优质之材
                                                                                                            3. 在比较之中选材 :::
                                                                                                          1. 通讯的表达
                                                                                                          详情
                                                                                                          • 通讯的表达技巧
                                                                                                            1. 紧扣主题进行表达
                                                                                                            2. 有层次感地进行表达。
                                                                                                            3. 富于特色地进行表达
                                                                                                            • 通讯表达的创新
                                                                                                              1. 在思维上求新
                                                                                                              2. 在文体上求新
                                                                                                              3. 在选材上求新
                                                                                                              4. 在语言上求新

                                                                                                          # 室外实践 1-通讯写作

                                                                                                          • 自主收集材料，完成一篇人物通讯/事件通讯/风貌通讯。
                                                                                                            1. 类别说明
                                                                                                              • 人物通讯：一人一生，为人物全面立传；一个人的一个或几个侧向；人物群像。
                                                                                                              • 事件通讯：突发性事件；在社会上产生较大影响的预知事件；反映社会精神风貌的小故事。
                                                                                                              • 风貌通讯：综合报道某个地区的今昔变化和新的建设成就；报道某地风土人情、人的精神面貌；报道历史文化遗产。
                                                                                                            2. 写作要求
                                                                                                              • 确定主题：确立体现时代精神，反映人物和实务、本质和规律、深刻的主题；
                                                                                                              • 选好材料：选择能支撑主题的典型材料；写好细节，注重情节事实；
                                                                                                              • 安排结构：纵式结构、横式结构、纵横式结构、递进式结构等。
