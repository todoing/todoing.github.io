    # 什么是马克思主义

    # 创立者：马克思和恩格斯

    # 马克思（1818-1883） 恩格斯（1820-1895）

    马克思胡子一把一把的（1818）

    1883 马克思归山，1895 恩格斯退伍

    生活时代：19世纪20年代到19世纪末

    1848《共产党宣言》标志着马克思主义的诞生

    # 陈望道，翻译《共产党宣言》第一人

    # 马克思主义分类

    伯恩斯坦的马克思主义

    列宁的马克思主义

    西方的马克思 主义

    佛洛依德的马克思主义

    存在主义的马克思主义

    中国的马克思主义——毛泽东思想 、习近平新时代中国特色社会主义思想（21世纪的马克思主义，当代中国的马克思主义）

    正统的马克思主义：中国的马克思主义

    # 马克思主义是不是洋主义

    马克思主义是洋主义，创立者 马恩 德国

    马克思主义也不是洋主义，发展——实现了中国化——正统：中国化的马克思主义
