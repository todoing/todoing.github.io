                      # 第一章：新媒体概述

                      # 一、了解传播媒介

                      1. 传播媒介的概念

                      • 指介于传播者与接受者之间的用以负载、传递、延伸、扩大特定符号的物质实体

                      • 核心构成要素

                        • 物体 - 物体实体是构成传播媒介的前提条件
                        • 符号 - 符号时构成传播媒介的第二要素
                        • 信息 - 信息是传播者与接受者发生联系、形成互动的诱因和前提

                      • 麦克卢汉的媒介理论

                      媒介类型清晰度信息量参与度举例
                      热媒介高大低照片、象形文字、无声电影、报刊等
                      冷媒介低小高漫画、拼音文字、有声电影、电视

                      1. 传播媒介的发展

                        • 人类社会的四次传播革命
                          1. 文字的发明和使用
                          2. 印刷术的发明
                          3. 电报的发明
                          4. 互联网技术的推广以及由此带来的新媒体的勃兴
                        • 印刷术传播媒介
                          • 报纸
                          • 杂志
                        • 电子类传播媒介
                          • 广播
                          • 电视
                          • 互联网
                            • 本质上是以 个人为中心的传播技术 ，具有天然的 反中心 取向

                      2. 传播媒介的类型和特点

                        1. 类型
                          • 按媒介出现历史顺序：符号媒介、语言媒介、文字媒介、电子媒介、网络媒介
                          • 按传播对象：个体传播媒介、大众传播媒介
                          • 按传播目的：公益性媒介、营利性媒介
                          • 按作用感官：听觉媒介、视觉媒介、视听媒介
                        2. 特点
                          • 实体性
                            • 报纸、杂志、电视等
                          • 扩张性
                            • 影响、感化、教化他人
                          • 还原性
                            • 不对接受的原始符号做扭曲、变形处理
                          • 中介性
                            • 介于传播者和接受者之间
                          • 负载性
                            • 媒介实体、符号和信息的混杂物

                      3. 新传播媒介

                        • 这是一个相对概念
                        • “ 新 ”在哪里
                          • 传播主体的多元化
                          • 传播内容和形式的多样化
                          • 传播行为的主动性
                          • 新的传播效力

                      # 二、互联网改变世界

                      1. 网络丰富日常生活
                        • 信息传递和交流变得更加便捷
                        • 打破传统人际交往的时空限制，促进了人际交往
                        • 让生活变得便利和丰富多彩
                      2. 网络推动社会进步
                        • 为经济发展注入新的活力
                        • 促进民主政治的进步
                        • 为文化传播和科技创新搭建新平台
                      3. 网络是把双刃剑
                        • 网络中有一些虚假的、不良的信息
                        • 沉迷于网络，影响学习、工作生活
                        • 个人隐私容易被侵犯

                      # 三、新媒体的界定

                      1. 新媒体的概念
                        • 利用数字技术、网络技术、通过互联网、宽带局域网、无线通信网、卫星等渠道，以及计算机、手机、数字电视等终端，向用户提供信息和娱乐服务的传播形态
                      2. 新媒体的构成要素
                        • 技术基础
                          • 数字、网络技术、有线与无线通道的传送
                        • 呈现方式
                          • 多媒体、跨媒体、跨时空、互动性
                        • 传播范围
                          • 全天候、全覆盖
                        • 创新性
                          • 技术、运营、商品、服务、商业模式
                        • 媒介融合

                      # 四、新媒体的类型

                      • 按传播类型
                        • 基于互联网的新媒体
                        • 基于数字广播网络的新媒体
                        • 基于无线网络的新媒体
                        • 基于融合网络的新媒体
                      • 按传播媒介
                        • 网络新媒体
                        • 手机新媒体
                        • 新型电视新媒体
                        • 其他新媒体
                      • 按传播形态
                        • 微博
                        • QQ
                        • 微信
                        • 网络直播
                        • 短视频
                        • 移动新闻客户端
                        • 自媒体
                        • 数字电视

                      # 五、新媒体的特征

                      1. 传播特征
                        • 主要依赖于数字化技术
                        • 由单向线性传播转变为交互性传播
                        • 传播行为更加个性化
                        • 传播成本降低
                        • 传播更新速度快
                        • 从单一传播到融合传播
                        • 实时发布和异步接收
                        • 共享海量的传播信息
                        • 能形成特定的效应
                      2. 产业特征
                        • 产业方式多样化
                        • 具有兼容性的技术
                        • 分众传播
                        • 适应扁平化需求
                        • 新媒体与传统媒体互相借力实现共赢
                      3. 舆论特征
                        • 扩展了传播空间
                          • 自动产生议题
                          • 缩短了舆论形成的周期
                      4. 文化特征
                        • 内容和形式的更新换代
                        • 遵守规则
                        • 平民文化
                        • 全球化扩展与不同文化的兼容
                        • 大众娱乐化
                        • 文化的双面性
